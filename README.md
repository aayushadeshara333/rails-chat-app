- One to one chat

- Real time user status
- Cross platform

## Run Locally

Clone the project

```bash
  git clone https://github.com/aayushadeshara333/rails-chat-app.git
```

Go to the project directory

```bash
  cd rails-chat-app
```

Install dependencies

```bash
  bundle
```

Start the server

```bash
  bin/rails start
```

That's it! You are good to go.
